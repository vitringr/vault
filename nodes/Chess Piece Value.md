# Chess Piece Value

Standard evaluation of chess pieces.

---

Useful as a rough guide when evaluating a position. However, the true value of a piece can vary significantly depending on the position.

| Piece  | Value      |
| ------ | ---------- |
| Pawn   | `1`        |
| Knight | `3`        |
| Bishop | `3`        |
| Rook   | `5`        |
| Queen  | `9`        |
| King   | `infinite` |
