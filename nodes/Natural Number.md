# Natural Number

(Counting Number)

Any positive [[Integer]] from `1` (or `0`) upward.

---

Numbers like (`1`, `2`, `3`, ...), often used for counting and ordering.

**Notation**: `N = {1, 2, 3, ...}`

Note that some systems include `0` to the set of natural numbers.
