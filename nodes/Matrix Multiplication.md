# Matrix Multiplication

[[Matrix]] multiplication intuition.

---

## Matrix and Vector

In 2D:

```
┌───┬───┐   ┌───┐
│ a │ b │   │ x │
├───┼───┤ × ├───┤
│ c │ d │   │ y │
└───┴───┘   └───┘

───────────────────

  ┌───┐     ┌───┐
  │ a │     │ b │
x ├───┤ + y ├───┤
  │ c │     │ d │
  └───┘     └───┘

───────────────────

<(xa + yb), (xc + yd)>

```

In 3D:

```
┌───┬───┬───┐   ┌───┐
│ a │ b │ c │   │ x │
├───┼───┼───┤   ├───┤
│ d │ e │ f │ × │ y │
├───┼───┼───┤   ├───┤
│ g │ h │ i │   │ z │
└───┴───┴───┘   └───┘

───────────────────────────

  ┌───┐     ┌───┐     ┌───┐
  │ a │     │ b │     │ c │
  ├───┤     ├───┤     ├───┤
x │ d │ + y │ e │ + z │ f │
  ├───┤     ├───┤     ├───┤
  │ g │     │ h │     │ i │
  └───┘     └───┘     └───┘

───────────────────────────

<(xa + yb + zc), (xd + ye + zf), (xg + yh + zi)>

```

## Multiple Matrices

In 2D:

```
┌───┬───┐   ┌───┬───┐
│ a │ b │   │ e │ f │
├───┼───┤ × ├───┼───┤
│ c │ d │   │ g │ h │
└───┴───┘   └───┴───┘

─────────────────────────────────────────────────

┌───┬───┐ ┌───┐     ┌───┐     ┌───┐   ┌─────────┐
│ a │ b │ │ e │     │ a │     │ b │   │ ea + gb │
├───┼───┤ ├───┤ = e ├───┤ + g ├───┤ = ├─────────┤
│ c │ d │ │ g │     │ c │     │ d │   │ ec + gd │
└───┴───┘ └───┘     └───┘     └───┘   └─────────┘

┌───┬───┐ ┌───┐     ┌───┐     ┌───┐   ┌─────────┐
│ a │ b │ │ f │     │ a │     │ b │   │ fa + hb │
├───┼───┤ ├───┤ = f ├───┤ + h ├───┤ = ├─────────┤
│ c │ d │ │ h │     │ c │     │ d │   │ fc + hd │
└───┴───┘ └───┘     └───┘     └───┘   └─────────┘

─────────────────────────────────────────────────

┌─────────┬─────────┐
│ ea + gb │ fa + hb │
├─────────┼─────────┤
│ ec + gd │ fc + hd │
└─────────┴─────────┘
```

In 3D:

```
┌───┬───┬───┐   ┌───┬───┬───┐
│ a │ b │ c │   │ 1 │ 2 │ 3 │
├───┼───┼───┤   ├───┼───┼───┤
│ d │ e │ f │ × │ 4 │ 5 │ 6 │
├───┼───┼───┤   ├───┼───┼───┤
│ g │ h │ i │   │ 7 │ 8 │ 9 │
└───┴───┴───┘   └───┴───┴───┘

──────────────────────────────────────────────────────────────────

┌───┬───┬───┐   ┌───┐     ┌───┐   ┌───┐   ┌───┐   ┌──────────────┐
│ a │ b │ c │   │ 1 │     │ a │   │ b │   │ c │   │ 1a + 4b + 7c │
├───┼───┼───┤   ├───┤     ├───┤   ├───┤   ├───┤   ├──────────────┤
│ d │ e │ f │ × │ 4 │ = 1 │ d │ 4 │ e │ 7 │ f │ = │ 1d + 4e + 7f │
├───┼───┼───┤   ├───┤     ├───┤   ├───┤   ├───┤   ├──────────────┤
│ g │ h │ i │   │ 7 │     │ g │   │ h │   │ i │   │ 1g + 4h + 7i │
└───┴───┴───┘   └───┘     └───┘   └───┘   └───┘   └──────────────┘

┌───┬───┬───┐   ┌───┐     ┌───┐   ┌───┐   ┌───┐   ┌──────────────┐
│ a │ b │ c │   │ 2 │     │ a │   │ b │   │ c │   │ 2a + 5b + 8c │
├───┼───┼───┤   ├───┤     ├───┤   ├───┤   ├───┤   ├──────────────┤
│ d │ e │ f │ × │ 5 │ = 2 │ d │ 5 │ e │ 8 │ f │ = │ 2d + 5e + 8f │
├───┼───┼───┤   ├───┤     ├───┤   ├───┤   ├───┤   ├──────────────┤
│ g │ h │ i │   │ 8 │     │ g │   │ h │   │ i │   │ 2g + 5h + 8i │
└───┴───┴───┘   └───┘     └───┘   └───┘   └───┘   └──────────────┘

┌───┬───┬───┐   ┌───┐     ┌───┐   ┌───┐   ┌───┐   ┌──────────────┐
│ a │ b │ c │   │ 3 │     │ a │   │ b │   │ c │   │ 3a + 6b + 9c │
├───┼───┼───┤   ├───┤     ├───┤   ├───┤   ├───┤   ├──────────────┤
│ d │ e │ f │ × │ 6 │ = 3 │ d │ 6 │ e │ 9 │ f │ = │ 3d + 6e + 9f │
├───┼───┼───┤   ├───┤     ├───┤   ├───┤   ├───┤   ├──────────────┤
│ g │ h │ i │   │ 9 │     │ g │   │ h │   │ i │   │ 3g + 6h + 9i │
└───┴───┴───┘   └───┘     └───┘   └───┘   └───┘   └──────────────┘

──────────────────────────────────────────────────────────────────

┌──────────────┬──────────────┬──────────────┐
│ 1a + 4b + 7c │ 2a + 5b + 8c │ 3a + 6b + 9c │
├──────────────┼──────────────┼──────────────┤
│ 1d + 4e + 7f │ 2d + 5e + 8f │ 3d + 6e + 9f │
├──────────────┼──────────────┼──────────────┤
│ 1g + 4h + 7i │ 2g + 5h + 8i │ 3g + 6h + 9i │
└──────────────┴──────────────┴──────────────┘

```
