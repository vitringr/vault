## Multiple Matrices

```
┌───┬───┐   ┌───┬───┐
│ a │ b │   │ e │ f │
├───┼───┤ * ├───┼───┤
│ c │ d │   │ g │ h │
└───┴───┘   └───┴───┘

─────────────────────────────────────────────────

┌───┬───┐ ┌───┐     ┌───┐     ┌───┐   ┌─────────┐
│ a │ b │ │ e │     │ a │     │ b │   │ ea + gb │
├───┼───┤ ├───┤ = e ├───┤ + g ├───┤ = ├─────────┤
│ c │ d │ │ g │     │ c │     │ d │   │ ec + gd │
└───┴───┘ └───┘     └───┘     └───┘   └─────────┘

┌───┬───┐ ┌───┐     ┌───┐     ┌───┐   ┌─────────┐
│ a │ b │ │ f │     │ a │     │ b │   │ fa + hb │
├───┼───┤ ├───┤ = f ├───┤ + h ├───┤ = ├─────────┤
│ c │ d │ │ h │     │ c │     │ d │   │ fc + hd │
└───┴───┘ └───┘     └───┘     └───┘   └─────────┘

─────────────────────────────────────────────────

┌─────────┬─────────┐
│ ea + gb │ fa + hb │
├─────────┼─────────┤
│ ec + gd │ fc + hd │
└─────────┴─────────┘
```
