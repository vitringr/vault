# C Variables

Variables in the C programming language.

---

| Data Type | Bytes      | Description                     |
| --------- | ---------- | ------------------------------- |
| `int`     | `2` or `4` | Integer                         |
| `float`   | `4`        | Single-precision floating point |
| `double`  | `8`        | Double-precision floating point |
| `char`    | `1`        | Single character                |
