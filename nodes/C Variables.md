# C Variables

Variables in the C programming language.

---

See [[C Data Types]]

```
type name;
type name = value;
```

## Name Rules

- Can contain letters, digits, and underscores.
- Must start with a letter or underscore "`_`".
- Cannot use [[C Keywords]].
- Case-sensitive.
