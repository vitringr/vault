# TSX

Syntax extension allowing for markup-like code in TypeScript.

```tsx
const SomeComponent = () => {
  <div>
    <h1>Header</h1>
    <p>paragraph</p>
    <canvas />
  </div>;
};
```

File extension: `.tsx`
