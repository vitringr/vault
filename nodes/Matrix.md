# Matrix

Rectangular arrangement of symbols in rows and columns.

---

Example matrices:

┌───┬───┬───┐  ┌───┬───┐  ┌───┬───┐
│ a │ b │ c │  │ a │ b │  │ a │ b │
├───┼───┼───┤  ├───┼───┤  ├───┼───┤
│ d │ e │ f │  │ c │ d │  │ c │ d │
├───┼───┼───┤  ├───┼───┤  └───┴───┘
│ g │ h │ i │  │ e │ f │
└───┴───┴───┘  └───┴───┘

**Order**: The _dimension_ or _order_ of a matrix is given by the number of rows times the number of columns.
