# Arithmetic Sequence

(Arithmetic Progression)

Sequence such that each term differs from the previous one by a constant value.

Examples:

`1, 2, 3, 4, 5, 6...`

`1, 1, 1, 1, 1, 1...`

`1, 0, -1, -2, -3, -4...`

`0, 33, 66, 99, 132, 165...`
